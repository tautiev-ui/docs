# APP FLOW: ConsumerShield User Journeys & Wireframes

## 🎯 Overview: Key User Flows

This document maps all primary user journeys through ConsumerShield platform.

**Primary Flows:**
1. Awareness → Consultation Booking → Payment
2. Document Upload → AI Analysis → Claim Generation
3. Case Status Tracking
4. Subscription Signup

- **Mobile** (< 640px) - Vertical stacking, touch-friendly buttons
- **Tablet** (640-1024px) - 2-column layouts
- **Desktop** (> 1024px) - Full layouts

**Mobile Optimizations:**
- Large touch targets (44×44px minimum)
- Full-width buttons
- Simplified forms (one field per row)
- Bottom navigation bar instead of top
